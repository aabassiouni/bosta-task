# BostaStore

## Running the project

Prerequisites:

- Node.js LTS
- pnpm v9.15.3

```sh
pnpm install
pnpm dev
```
## Tech Stack

- [React](https://reactjs.org/)
- [React Router](https://reactrouter.com/)
- [React Query](https://tanstack.com/query/latest)
- [Tailwind CSS](https://tailwindcss.com/)
- [Vite](https://vitejs.dev/)
